# cat_dog_classification
## 📣Introduction

一个非常easy的项目，仅仅只是为了通过猫狗分类实例熟悉基本的Tensorflow接口🎇。等之后兴致来潮就updating🤓~

## 💊Dependence

* Python 3.8
* Tensorflow 2.2.0
* Numpy 1.20.1
* Matplotlib 3.7.1

## 🧨Usage

数据集下载地址：https://www.kaggle.com/datasets/tongpython/cat-and-dog

训练集、验证集、测试集比例：8 : 1 : 1

数据集目录结构：

```bash
data/
├── test
│   ├── cats
│   │   ├── cat.1.jpg
│   │   ├── cat.2.jpg
│   │   ├── .
│   │   ├── .
│   └── dogs
├── train
│   ├── cats
│   └── dogs
└── val
    ├── cats
    └── dogs
```
